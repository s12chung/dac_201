Also, I found these resources most useful as a reference:

Typographic Design Patterns and Best Practices

This site provides various statistics on web design, such as average font size, most common font and width ratios.

Revised Font Stack

This site provides guidance and recommendations for font stacks, a prioritized list of fonts for website use. It includes statistics on pre-installed fonts on different operating systems and a list of recommend font stacks. Explanations and example usages are also provided for six of the font stacks.

Some Typographic Rules to Remember

Adapted from "The Mac Is Not a Typewriter" by Robin Williams, this is a highly comprehensive list of typographic tips.

Firebug

Firebug is a great plug-in for your internet browser to see the code behind the website. It can be used to analyze the typography and layout of any website.
