The size and weight (or styling) of the text can change the effectiveness of a font.

Size

Selecting a font size can be very ambiguous. The most popular size for headlines is 18 to 20 pixels and 24 to 26 pixels. The body usually has font sizes of 12 to 14 pixels.

When choosing a font size, follow standard scaling, a set of pre-defined sizes, to increase harmony between type. You can see all the scaling sizes in the picture below.

Weight

There are two reasons that drive the selection of weight:

  • Aesthetic, adding style or follow composition
  • Content, the content dictates importance or mood, such as quotations

Using these reasons as a guide, one can use many ways to increase the weight of a font, bolding, capitalization or a combination of methods.

Bolding is preferred over most methods to add weight because it's clean, noticeable and readable. Underlining adds noise to the design and are often associated with links. Italicization can be hard to see. Capitalized letters are harder to read because the letters have more similar height and look.

Different colors can be used to add or remove weight. To de-emphasize text, limit the contrast of the text to its background without changing the text's color. In fact, most web sites have a white background and text color a bit lighter than pure black.

Try to limit combinations and avoid overusing weighting methods.

Implementation

To implement font size, simply use the font-size CSS property, as shown below.

.some_class {
   font-size: 1em;
}

Weight implementation varies for each method, see the following code for weighting methods in CSS.

.some_class {
   font-weight: bold;
   text-decoration: underline;
   font-style: italic;
   font-variant:small-caps;
   text-transform: uppercase;
   color: #222222; /* this is dark gray */
}
