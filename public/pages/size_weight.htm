The size and weight (or styling) of the text can affect it's impact on the user. This page will go through tips on how to select the size and weight of your text.

Size

Selecting a font size can be very ambiguous. The most popular size for headlines are between 18 to 20 pixels and 24 to 26 pixels. The body usually has font sizes of 12 to 14 pixels.

When choosing a font size, follow standard scaling, a set of pre-defined sizes, to increase harmony between type. It's kind of like the Fibonacci numbers, its naturally appealing. You can see all the scaling sizes in the picture below.

Weight

There are two reasons that drive the selection of weight:

  • Aesthetic, adding style or to follow composition
  • Content, the content dictates importance or a mood, such as quotations requiring italicization

Using these reasons as a guide, one can use many methods to increase the weight of a font, such as bolding, capitalization or a combination of methods.

When selecting a method, remember that bold is preferred to add weight because it's clean, noticeable and readable. Underlining adds noise to the design and are often associated with links. Italicization can be hard to see. Capitalized letters are harder to read because the letters have similar heights and look.

Different colors can be used to add or remove weight. Decreasing the opacity of the text color is often used to de-emphasize text. In fact, most web sites have a white background and text color a bit lighter than pure black.

With all these methods however, you should try to limit combinations and avoid overusing weighting methods because overuse makes strong weights look normal, different weighted text compete against each other and the combinations might look the same. In other words, your text will look messy. Simplicity is key.

Implementation

To implement font size, simply use the font-size CSS property, as shown below.

.some_class {
   font-size: 1em;
}

Weight implementation varies for each method, see the following code for weighting methods in CSS.

.some_class {
   font-weight: bold;
   text-decoration: underline;
   font-style: italic;
   font-variant:small-caps;
   text-transform: uppercase;
   color: #222222; /* this is dark gray */
}
