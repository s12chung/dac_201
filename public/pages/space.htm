Whitespace de-clutters type by distinguishing elements into separate groups creating relationships between elements. Whitespace also makes type more readable, as it has been found that adding whitespace can increase comprehension by 20%. The layout of text also influences the user satisfaction of the web site.

Measure

The Measure is the width of the body. The measure should be about 55 to 85 characters in length (including spaces) to make it easier to read. However, most sites have a measure from 73 and 90 characters.

Implementing measure in your web site varies upon the situation. However, a good guideline is to adopt a grid system to help, such as the 960 grid system.

Leading

Leading is the vertical space between lines of text. It is recommended that the leading should be wider than the word spacing. This practice makes it easier for the eye to go along the line, instead of down the lines.

Statistically, the average line height to body copy font size ratio (line height รท body copy font size) on web sites is 1.48 (1.5 is recommended).

The leading should increase proportionally to the measure. Thus, small measure, less leading. Wide measure, more leading. The average ratio between the measure and leading is 27.8. However, this number is considered fairly large for typographers.

Also note that the space between paragraphs should be about 75% of the line height for more vertical rhythm.

Implementation

To add leading in CSS, use the line-height property. To change the space between paragraphs, use the margin-top property on your p (paragraph) tags. For example:

p {
   line-height: 150%;
   margin-top: 12px; /* assuming the font size is 16px */
}
